============================
CHANGING DEFAUL COLOR SCHEME
============================

Please follow the steps below in order to change the default color scheme: 

1. Open index.html, find line 36 and change the <body> class to .body-blue, .body-green, .body-asphalt, or .body-red accordingly. 
2. Remove the following code to disable color change option: 

    <!-- Color Options -->
    <div class="color-options">
      <div class="option red" data-color="red"></div>
      <div class="option blue" data-color="blue"></div>
      <div class="option green" data-color="green"></div>
      <div class="option asphalt" data-color="asphalt"></div>
    </div>
